# team_bechdel_446
Federal Reserve Bank of New York: Survey of Consumer Expectations
In this project there are multiple different datasets used, as well as multiple different folders that can be accessed.
The "Bechdel Group.pdf" is our explanation of the project, the tools and methodologies used throughout the project, and the steps we took to make the data useable.
The "Bechdel.pptx" is our main presentation which contains all of our findings, presented at a former time and place.
"New Bechdel Tableau Data 2.xlsx" is the filtered data that we used for the project. Cleaned up by hand to ensure proper function use in Tableau.
"Visuals for Project.pdf" has all of our atached Tableau worksheet screenshots
